# 🖼️ Image Tampering Technique using COCO Dataset

## 📌 Project Overview
This project focuses on generating and studying **image tampering techniques** for research in image forensics. The objective is to create a **tampered version of the COCO dataset** using different manipulation techniques and analyze the characteristics of forged images.

Image tampering is commonly used in misinformation, digital fraud, and manipulated media. Detecting such manipulations is an important problem in computer vision and digital forensics.

---

## 🧠 Tampering Techniques Implemented

### 🔁 1. Copy-Move Tampering
Copy-move tampering is a manipulation technique where a part of an image is copied and pasted within the same image.

**Purpose:**
- 🔹 Hide objects  
- 🔹 Duplicate objects  
- 🔹 Modify scene content  

**Challenges:**
- ⚠️ Detecting duplicated regions  
- ⚠️ Handling rotation and scaling  
- ⚠️ Maintaining realistic blending  

---

### ✂️ 2. Copy-Paste Tampering
In copy-paste tampering, a region from one image is copied and pasted into another image.

**Purpose:**
- 🔹 Insert fake objects  
- 🔹 Modify scene composition  
- 🔹 Create manipulated media  

**Challenges:**
- ⚠️ Boundary artifacts  
- ⚠️ Lighting mismatch  
- ⚠️ Texture and color inconsistencies  

---

## 🗂 Dataset

The experiments are performed using the **COCO (Common Objects in Context) Dataset**.

Dataset contains:
- 📸 High quality natural images  
- 🧩 Multiple object categories  
- 🌍 Realistic scenes for manipulation experiments  

A **tampered dataset** is generated by applying copy-move and copy-paste techniques on original COCO images.

## 🎨 Blending Techniques for Image Tampering

To create realistic tampered images, different **blending techniques** are used.  
These techniques help reduce visible artifacts and make the manipulated regions appear natural.

The project implements a **Blending Engine** that supports multiple blending methods.

### Supported Blending Methods

| Method | Description |
|------|-------------|
| 🔹 **Alpha Blending** | Simple pixel-wise blending using a binary mask |
| 🔹 **Gaussian Blending** | Uses Gaussian smoothing on mask edges for smoother transitions |
| 🔹 **Poisson Blending** | Advanced gradient-based blending using OpenCV seamless cloning |
| 🔹 **Histogram Matching Blend** | Adjusts foreground color statistics to match the background |

## 📊 Benchmarking Metrics

The tampered images are evaluated using the following **quantitative metrics**:

| Metric | Description |
|------|-------------|
| **PSNR** (Peak Signal-to-Noise Ratio) | Measures distortion between original and tampered images |
| **SSIM** (Structural Similarity Index) | Evaluates structural similarity |
| **NCC** (Normalized Cross-Correlation) | Measures similarity between image patches |
| **Edge Artifact Score** | Detects unnatural edges around manipulated regions |

These metrics help assess **tampering realism and detectability**.
